# Resume Screening Automation

## Objective
To automate initial resume screening by extracting and filtering candidate
profiles based on required skills using Python.

## Tools Used
- Python
- Pandas
- Regex

## Key Work
- Parsed resume text to extract skills
- Applied filtering logic based on job requirements
- Automated basic shortlisting process

## Outcome
Reduced manual resume screening effort and improved shortlisting efficiency.
